# Network Port Scanner GUI

This project is a Python-based Network Port Scanner with a graphical user interface (GUI) built using Tkinter.

## Features
- Scan open ports on a target IP or hostname
- Multi-threaded scanning for faster performance
- Simple and user-friendly GUI
- Displays open ports clearly

## Technologies Used
- Python
- Tkinter
- Socket Programming
- Multithreading

## How to Run
1. Install Python
2. Run the file:
   python portscannergui.py
